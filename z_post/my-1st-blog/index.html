Homebrew Cask

Homebrew Cask是Homebrew的扩展（已内置），使用它可以安装brew命令不能安装的Mac App，使用它可以安装不上架 Mac App Store 的应用，其核心是简化 macOS 上的软件安装和管理过程，通过极其简洁的一条命令就可以优雅、快速的搞定macOS软件包的安装（可以类比一键安装）和管理。

换句话说：在Mac上安装常用软件包，大部分情况下只需要使用 Homebrew Cask 就足够了，无需Homebrew出手。

Homebrew 通过提供友好的CLI工作流来管理作为二进制文件分发的macOS应用程序。

Homebrew 在 2009 年由马克斯·霍威尔（Max Howell）写成，它在 GitHub 上拥有大量贡献者，目前仍处于活跃状态。

Homebrew Cask由社区进行维护，因此它有更多，更丰富的软件。

Homebrew 中文含义为自制、自酿酒，Cask 中文含义为桶、木桶，桶装酒是一种成品，也就是说每一个 homebrew cask 都可以直接使用的，比如 Atom 的 Cask 名称为 atom，那么就可以使用如下命令安装：

brew cask install atom

Homebrew 和Homebrew cask的区别

  • Homebrew 是通过直接下载源码，并自动执行软件包安装。
  • Homebrew 会将软件包安装到独立目录，并将其文件软链接至 /usr/local 。
  • Homebrew cask 则是下载已经编译好的软件包（.dmg/.pkg），并自动执行软件包安装，如QQ，Chrome，evernote等。
  • Homebrew cask安装软件时自动创建软连接到Application目录，这样在Launchpad中也能查看到安装的软件，方便启动软件。
  • 两者都可以自动配置环境变量，以命令来安装、卸载、更新软件，很方便。

Mac App Store的问题

  • Mac App Store生态圈远不完善，审核流程过长，限制太多，维护成本过高让很多应用开发者被迫离开，如MPlayerX已经宣布不在更新Mac App Store上的版本。
  • Mac App Store的新版本更新不及时而且周期长，如Alfred 2.0已经出了很久，但在Mac App Store上还是1.2版本。
  • 在天朝Store的速度XXOO。

Homebrew cask对比Mac App Store下载安装的优势

  • Homebrew Cask 安装App，无需打开Mac App Store。
  • Homebrew Cask 让非 Mac App Store 应用的安装更轻松。
  • Homebrew Cask 安装的 App 也是从它的软件官网下载，Homebrew 只是做了整合，这使得它对于常用 App 的支持更全面，更新也更迅速。
  • Homebrew cask安装Mac APP全部过程只需要一条命令搞定，如果你想同时安装多个软件包，这种方法的优势更加明显（它还是只需要一条命令）。

Homebrew cask对比从官网下载安装的优势

  • 通过 Homebrew 或者 Homebrew Cask 安装的软件都会集中在一个目录下面，再由 Homebrew 将相关软件软链接到相关路径。举个例子，如果当我们访问系统中的 A 文件时，系统都会将访问路径导向 B 文件对应的路径，那么 A 文件就是 B 文件的软链接，这也是 Homebrew 的原理。
  • 相比我们自行去官网下载的 App 安装包，有时在「桌面」有时在「下载」文件夹的情况，使用 Homebrew 我们无需担心文件的位置，所有安装包都会放在一处，这对之后的管理提供了很大便利。
